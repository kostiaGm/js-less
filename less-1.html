Javascrip может находится внутри тегов script внутри документа и подключаться из файла

Javascript внутри страницы 
    <script type="text/javascript">
    /**
        Код javascript
    */
    </script>
Подключаем из файла. Внутри тегов ничего не указываем!!! 
    <script type="text/javascript" src="some-file.js"></script>

Скрипты лучше указывать над тегом </body> потому, что скрипты будут вызываться после загрузки всего документа.

Обовсех тегах, которые находятся ниже тегов script, Javascript ничего не узнает. Однако, в различных Javascript фреймвоках и библиотеках, реализован функционал, позволяющий вызывать скрипты на более высоком уровне HTML документа.
